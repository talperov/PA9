Before launching game you must ensure you have to correct include directories, linker directories, and SFML version.

This program uses SFML 2.5.1 Visual C++ 15 (2017) 64-bit. You can download SFML at sfml-dev.org

To ensure you have the correct include and linker directories, follow the below instructions:

1. Under the solution explorer, right click PA9_GROUP and select properties
2. Navigate to the C/C++ menu and then click general. Ensure that the "Additional Include Directories" has an absolute path to your SFML-2.5.1\include folder
3. Navigate to the Linker menu and then click general. Ensure that the "Additional Library Directories" has an absolute path to your SFML-2.5.1\lib folder

Playing the game:

Launch the game from VS Community and you will have two options, test or play.
Test will run test cases of various functions in our code
Play will let you play the game

This is a 2D fighting game between two players. WASD will move player 1 and the arrow keys will move player 2.
Player 1 attacks with the space bar, player 2 attacks with right control.
Enjoy!
